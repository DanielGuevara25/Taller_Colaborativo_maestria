# Segmentación de Clientes con Aprendizaje No Supervisado

## Descripción
Este proyecto aplica técnicas de aprendizaje no supervisado para segmentar clientes de un centro comercial según variables como edad, género, ingreso anual y puntaje de gasto.

## Dataset
Se utilizó el dataset Mall Customer Segmentation, disponible en Kaggle.

## Técnicas aplicadas
- K-means
- DBSCAN
- PCA
- t-SNE

## Metodología
1. Carga y limpieza de datos.
2. Análisis exploratorio.
3. Escalamiento de variables.
4. Aplicación de modelos de clustering.
5. Visualización con PCA y t-SNE.
6. Interpretación de perfiles.

## Conclusión
K-means permitió identificar segmentos comerciales claros, mientras que DBSCAN ayudó a detectar posibles clientes atípicos. PCA y t-SNE facilitaron la visualización de los grupos encontrados.
